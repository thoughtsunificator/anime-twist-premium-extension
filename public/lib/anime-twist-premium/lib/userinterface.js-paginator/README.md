# userinterface.js-paginator [![Gitpod Ready-to-Code](https://img.shields.io/badge/Gitpod-ready--to--code-blue?logo=gitpod)](https://gitpod.io/#https://github.com/thoughtsunificator/userinterface.js-paginator)

Pagination system for userinterface.js

## Getting started

### Installing

Run ```git submodule add https://github.com/thoughtsunificator/userinterface.js-paginator.git lib/userinterface.js-paginator``` at the root of your project.

userinterface.js-paginator should be located in the `lib/` folder at the root of your project.
