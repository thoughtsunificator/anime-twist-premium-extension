# anime-twist-premium [![Gitpod Ready-to-Code](https://img.shields.io/badge/Gitpod-ready--to--code-blue?logo=gitpod)](https://gitpod.io/#https://github.com/thoughtsunificator/anime-twist-premium)

Anime Twist Premium is a userscript that allows you to add anime to a watch list so that you can easily retrieve them later.

## Getting started

### Installing

#### Extension

Header over to the [anime-twist-premium-extension](https://github.com/thoughtsunificator/anime-twist-premium-extension) for the extension.

#### Userscript

First you have to install an userscript manager:

[Violentmonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/) for Firefox.

[Violentmonkey](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag?hl=en) for Chrome.

Finally, install the userscript from [OpenUserJS](https://openuserjs.org/scripts/thoughtsunificator/anime-twist-premium).
